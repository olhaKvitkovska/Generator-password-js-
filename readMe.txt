Задача №5. Генератор пароля.

Напишите скрипт, который генерирует случайную строку. В строке должны быть маленькие и большие латинские буквы, цифры и подчеркивание.

Чтобы сгенерировать случайное число от 0 до max включительно используется формула Math.floor(Math.random( ) * (max+1)).
Для генерации случайного числа от min до max, включая и min и max используется формула Math.floor(Math.random( ) * (max - min + 1)) + min
